Aside from the obligatory update to 26.1.x, this update contains a couple of big re-considerations for the future of **Florum Sporum**, and as such is a little delayed. Thanks very much for your patience!!

# ➖ Removals
The panda sneeze mechanic from recent releases has been removed. My attitude towards game design has changed a lot and I decided it was very out of scope for what **Florum Sporum** tries to be.

# 🔧 Changes
- Spore blossoms now have 2D inventory sprites.
- There are no longer separate items for each growth stage. Placing the block will now default to the first growth stage, which you can advance with further bonemeal.
  - This is to reduce the amount of inventory clutter. Make sure to send me feedback on the GitHub issues page!

# 🐛 Fixes
- The Farmer's Delight cutting recipe no longer fails to load.
